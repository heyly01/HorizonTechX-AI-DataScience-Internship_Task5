# 🤖 AI Object Detection System

## 📌 Project Overview

This project was developed as part of the Horizon TechX AI & Data Science Internship.

AI Object Detection System is a Computer Vision project built using YOLO (You Only Look Once) and OpenCV to detect and track objects from images and real-time video input.

The system identifies objects, displays bounding boxes, confidence scores, and provides an interactive interface for object detection using AI-based vision techniques.

## 🚀 Features

- Real-time object detection
- Image-based object detection
- Webcam object tracking
- Bounding box visualization
- Object label identification
- Confidence score display
- Adjustable detection confidence
- Interactive Streamlit interface

## ⚙️ Installation & Setup

Clone the repository:

Install required dependencies:

pip install -r requirements.txt


Run the Streamlit application:

streamlit run app.py


For webcam detection:

python webcam_detection.py

## 🧠 Computer Vision Approach

Computer Vision enables machines to understand and analyze visual information from images and videos.

This project uses YOLOv8, a deep learning-based object detection model, to identify objects efficiently in real-time.

YOLO divides images into regions, detects objects, predicts labels, and provides confidence scores with bounding boxes.


## 🔍 Detection Capabilities

The system can detect multiple objects including:

- Person
- Vehicles
- Animals
- Daily objects
- Electronic items
- Common real-world objects

using a pre-trained YOLO model.
